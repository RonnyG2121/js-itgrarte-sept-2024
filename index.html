Bienvenida

¡Recibe una cordial bienvenida al curso de introducción a la programación con JavaScript! ¡Gracias a la fundación Itgrarte! En este curso, aprenderás la lógica de programación desde 0 y conocerás al lenguaje rey de la WEB. Lo que aprenderás:

  • Una breve historia sobre JavaScript
  • Conceptos sobre la programación
  • Variables y constantes
  • Operadores
  • Tipos de datos que maneja JavaScript
  • Estructuras de datos
  • Sentencias de control de flujo (condicionales y bucles)
  • Funciones

Requisitos para este curso

  • Manejo de Windows u otro sistema operativo
  • Tener conocimientos básicos de computación
  • Manejo de un editor de código, IDE o procesador de textos
  • Manejo de internet
  • Saber trabajar con Google ClassRoom

¿Qué necesito para aprobar?

  • Poder entender como funciona un algoritmo y como aplicarlo en la resolución de un problema
  • Buenas prácticas y saber trabajar con la división de código
  • Tener una asistencia al menos del 80% en las clases

En el curso las clases serán teóricas y prácticas. Razón por la cual el alumno tendrá que entregar trabajos prácticos y completar los cuestionarios en ClassRoom. Por lo que se recomienda al menos 5 horas semanales para resolver los ejercicios.

Al terminar el curso, tendremos un examen múltiple choice en ClassRoom y un ejercicio práctico que se deberá entregar en el tiempo estimado.

Pautas a seguir

  • El curso estará dividido en 6 clases y se compartirán por Google Meet los sábados, de las 10:00 A.M. a las 11:30 A.M. (Hora de Argentina)
  • Las clases serán grabadas para la revisión del alumno y repase lo que no entendió
  • El alumno tendrá que entregar trabajos prácticos durante las 6 clases, y deberá hacerlo en el tiempo estipulado por el mentor

Herramientas recomendadas a utilizar

  • Node.JS
  • Visual Studio Code
  • Extensión para el idioma español de Visual Studio Code
  • Complemento remote para NVDA (si lo usas)

El enlace al video de cada clase y los archivos de código de los temas serán añadidos en cada sección.
