Operadores, tipos de datos y estructuras

Operadores

Un lenguaje de programación de pleno derecho, necesita de manera obligatoria de unos signos para poder hacer operaciones variadas.

A estos solemos llamarlos operadores. Son los que nos permiten manipular y comparar los valores en las variables.

Nos permiten realizar cálculos matemáticos complejos en nuestros programas, dotándolos de una gran capacidad.

Los operadores que tenemos en JavaScript y en cualquier otro lenguaje son:

  • Operadores de Asignación: Estos proporcionan valores a las variables. Un operador de asignación asigna un valor a su operando izquierdo calculando en el valor de su operando derecho. El operador de asignación simple es igual ( =)
  • Operadores de comparación: comparan la coicidencia de valores en las variables. Un operador de comparación compara sus operandos y devuelve un valor lógico en función de si la comparación es verdadera ( true) o falsa ( false)
  • Operadorres aritméticos: Nos permiten realizar las operaciones matemáticas básicas. Un operador aritmético toma valores numéricos (ya sean literales o variables) como sus operandos y devuelve un solo valor numérico
  • Operadores lógicos: Evalúan si una determinada condición se cumple o no y si varias coiciden. Los operadores lógicos se utilizan normalmente con valores booleanos (lógicos)
  • Operadores de cadena: Permiten la concatenación de cadenas de textos
  • Operador condicional (ternario): Se usan para crear condicionales con una parte verdadera y una falsa
  • Operador coma (,): Se utiliza para la concatenación de variables con cadenas y parámetros en funciones
  • Operadores unarios: Nos permiten realizar operaciones con un único valor operando
  • Operadores relacionales: Son los que usamos para la evaluación de expresiones y normamente se usa el Septiembrer que (>) o Septiembrer o igual que (>=) y viceversa

Para más detalles visita: Documentación completa de los operadores y expresiones

Tipos de datos

JavaScript, al igual que otros lenguajes de programación cuenta con una variedad de tipos de datos y sus estructuras y una de las diferencias entre un lenguaje y otro es que estas pueden variar.

A causa de esto, cada lenguaje se especializa de cierta manera dependiendo de los tipos de datos que este pueda manejar.

JavaScript es un lenguaje débilmente tipado y dinámico. Las variables en JavaScript no están asociadas directamente con ningún tipo de valor en particular, y a cualquier variable se le puede asignar (y reasignar) valores de todos los tipos.

Los tipos de datos que conforman este lenguaje son:

  1. Undefined: Representa una variable que se ha creado, pero no tiene ningún valor
  2. Boolean: Se trata de 2 valores, true o false
  3. Number: Guarda números flotantes o enteros
  4. String: Almacena cadenas de textos
  5. BigInt: Se utiliza para almacenar números especialmente grandes
  6. Simbol: Crea identificadores únicos para los objetos
  7. Null: Representa la ausencia intencional de valores. Un valor nulo o vacío
  8. Object: Almacena una colección de datos distintos

Estructuras de datos

Las estructuras de datos son una forma de organizar los datos en un programa.

Esto facilita en gran manera el trabajo con los datos, ya que nos permiten una gestión más eficiente.

Dentro de las estructuras de datos tenemos:

  • Arrays: también llamadas arreglos, matrices o vectores, Permiten almacenar una serie de valores
  • Objetos: ya mencionados como un tipo de dato, también es una estructura muy importante. Nos permiten almacenar una serie de propiedades y valores de cualquier tipo

En JavaScript tenemos más estructuras avanzadas, tales como los sets o los maps. Estas no serán trabajadas en el curso ya que es de introducción.
